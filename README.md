## 📌 Overview

This project showcases a professional **n8n automation workflow** that automatically processes incoming emails, analyzes content using AI, applies business logic, and performs automated actions without manual intervention.

The workflow is designed to improve productivity, reduce repetitive work, and demonstrate practical workflow automation skills using modern no-code/low-code tools.

## 🚀 Getting Started

### 1️⃣ Clone Repository

```bash
git clone https://github.com/venkataramana01/n8n-email-automation.git
```

### 2️⃣ Import Workflow

Open n8n and navigate to:

```text
Workflows → Import from File
```

Select:

```text
workflow/workflow.json
```

### 3️⃣ Configure Credentials

Add your own:

* Gmail Credentials
* OpenAI API Key
* External API Keys
* Webhook URLs

### 4️⃣ Run Workflow

Execute the workflow and verify successful processing.
